Homework 3: Divide numbers


Write a JavaScript program to divide two numbers. The function should accept two numbers as parameters and should return a string with the sum. Call the function three times with the following numbers: 6888, 123; 3510, 234; 25, 5; and make sure to print out the results in the console or in an alert window



    function divideNumbers(num1, num2) {
        var sum = num1 / num2;
        return `${num1} divided by ${num2} is ${sum}.`;
    }
    
    var division = divideNumbers(6888, 123);
    console.log(division);
    
    console.log(divideNumbers(3510, 234));
    
    alert(divideNumbers(25, 5));

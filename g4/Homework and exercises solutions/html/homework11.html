Homework 11: Sort array


Write a JavaScript program to sort the items of an array.



    var array = [-3, 8, 7, 6, 5, -4, 3, 2, 1];

    for (var i = array.length; i > 0; i--) {
        for (var j = 1; j <= i; j++) {
            if (array[j - 1] > array[j]) {
                var temp = array[j - 1];
                array[j - 1] = array[j];
                array[j] = temp;
            }
        }
    }

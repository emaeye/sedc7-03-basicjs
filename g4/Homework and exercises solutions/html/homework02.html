Homework 2: Multiply numbers


Write a JavaScript program to multiply two numbers. The function should accept two numbers as parameters and should return the sum as a number. Call the function three times with the following numbers: 12, 1234; 65, 234; 25, 5; and make sure to print out the results in the console or in an alert window.



    function multiplyNumbers(num1, num2) {
        var sum = num1 * num2;
        return sum;
    }
    
    var multiplication = multiplyNumbers(12, 1234);
    console.log(multiplication);
    
    console.log(multiplyNumbers(65, 234));
    
    alert(multiplyNumbers(25, 5));

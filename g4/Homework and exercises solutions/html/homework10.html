Homework 10: Reverse array


Write a program in JavaScript to read n number of values in an array and display it in reverse order.



    var array = [];
    var n = 10;
    
    for (i = 1; i <= n; i++) {
        array.push(i);
    }
    
    var reversedArray = [];
    
    for (i = array.length; i > 0; i--) {
        reversedArray.push(i);
    }
    
    console.log(reversedArray);

Homework 19: Sum of squares 101 - 150


Write a JavaScript program to write the sum of squares of the numbers from 101 to 150.



    var sum = 0;

    for (var i = 101; i <= 150; i++) {
        var square = Math.pow(i, 2);
        sum += square;
    }
    
    console.log(`The sum of the squares is: ${sum}`);

Homework 9: Pyramid pattern


Write a JavaScript program to construct the following pattern, using a nested for loop.



    var numberOfRows = 5;
    var row = '';
    
    for (var i = 1; i <= numberOfRows; i++) {
        for ( var j = 1; j <= i; j++) {
            row += '* ';
        }
        console.log(`${row}\n`);
        row = '';
    }

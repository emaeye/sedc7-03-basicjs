Things you might going to find confusing in the exercise code:

console.log(something) - Log something in the console.

alert(something) - Trow a pop up screen in the browser and print the information inside the round parentheses.

document.writeln(something) - Print something directly on the screen (not often used).

prompt("message to the user") - Displays a dialog box that asks the visitor for input. It is often used to assing a value on a variable
trought input window direclty from the user. Prompt always returns a string.

parseInt("5") - Convert the value inside the round parentheses in integer. //result: 5.

Math.floor(2.8) - Rounds a number DOWNWARDS to the nearest integer. //result: 2.

isNaN("some text") - checks if the value inside the round parentheses is NOT A NUMBER, and return true or false. //result: true.

Methods used for array manipulation

array.length() - Returns the number of total items in the array.

array.push(x) - Add x at the end of the array.

array.pop() - Removes the last item of the array.

array.splice(X,Y) - Removes the item from the array on the X index, and cut Y number of elements infront of him.

array.sort() - Sorts the elements of the array in alphabetical order. If you want to sort numbers, you
need to add function(a,b){a - b} inside round parentheses in order to sort numbers by there value, else 11 will be
sorted before 2. Check this example and try this personally!

array.reverse() - Reverses the order of the elements in an array.

array.join() - Joins the elements of an array into a string, and returns the string. The elements will be separated
by a specified separator. Try this example in order to understand the separators.

array.split() - Split a string into an array of substrings, and returns the new array. The split method does not change the original string.
